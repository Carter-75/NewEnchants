# Custom Enchantment Blueprint (Minecraft 1.21.1 Fabric Datapack)

These 50 curated enchantments target Fabric 1.21.1 datapacks while staying close to vanilla balance. Each entry lists supported gear, level cap, per-level mechanics, and a shaped recipe for crafting the enchanted book (`minecraft:enchanted_book`). Item names reference canonical Minecraft content; all recipes lean on late-game drops, smithing templates, and trial-chamber loot.
